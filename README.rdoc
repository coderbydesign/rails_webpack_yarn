== Removing the Rails Asset Pipline in Favor of Webpack and Yarn

A translation of the steps laid out in this article: http://pixelatedworks.com/articles/replacing-the-rails-asset-pipeline-with-webpack-and-yarn/

* Adds webpack and yarn
* Removes front-end gem dependencies
* Updates asset configuration
* Versions assets for production

Local instructions:

* brew install node
* brew install yarn
* yarn global add webpack
* webpack
